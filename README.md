# popx-auth
Modern React mobile UI clone of PoPX app onboarding screens 
 Features
 Pixel-perfect mobile UI (360x640 viewport)

 4-screen flow with diagonal progress dots

 Signup form (name, phone, email, password, company, agency radio)

 Account profile with avatar badge

 Purple/orange theme matching design
 
 Responsive + smooth navigation

Tech Stack
jsx
React 18 • CSS Grid/Flexbox • useState Hooks • Mobile-First Design

Quick Start
bash
# Clone repo
git clone https://github.com/YOUR_USERNAME/popx-auth-ui-clone.git
cd popx-auth-ui-clone

# Install
npm install

# Run dev server
npm start
